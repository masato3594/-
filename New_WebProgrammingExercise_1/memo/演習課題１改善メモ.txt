★制作演習1改善メモ(最終版)2024/3/18

・web-fontの効かせ方をちゃんと調べる
・デザイン崩れを防ぐために親要素でちゃんと調整
・要素が重なるときに背景色が混ざらないようにする方法を調べる
⇒z-indexだけでいけるか？
.element-a {
    background-color: rgba(255, 0, 0, 0.5); /* 透明度を持つ背景色 */
    z-index: 1; /* 要素Aを要素Bの上に重ねる */
}

.element-b {
    background-color: rgba(0, 0, 255, 0.5); /* 透明度を持つ背景色 */
    z-index: 0; /* 要素Bを要素Aの下に配置 */
}
